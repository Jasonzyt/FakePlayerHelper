## Usage
1. Download `FakePlayerHelper-[Version].dll` from [Releases](https://github.com/Jasonzyt/FakePlayerHelper/releases)
2. Put the plugin into plugin folder of your plugin loader(or injector)
3. Run `bedrock_server.exe`
